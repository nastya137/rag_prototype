В этой директории предполагается хранение исходных документов,
которые используются для наполнения векторного хранилища (RAG).

## Требования к документам
- Форматы: PDF, DOCX, TXT
- Язык: русский

## Использование
1. Поместите документы в эту директорию
2. Запустите `get_document.py`
3. Векторная база будет создана автоматически
