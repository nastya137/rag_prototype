В директории /documents предполагается хранение исходных документов,
которые используются для наполнения векторного хранилища (RAG).

## Требования к документам
- Форматы: PDF, DOCX, TXT
- Язык: русский

## Использование
1. Поместите документы в директорию `/documents`
2. Запустите `get_document.py`
Важно: при первом запуске необходимо подключение к сети Интернет для загрузки модели!
3. Векторная база будет создана автоматически
4. Запустите `get_answer.py` 
5. Введите
